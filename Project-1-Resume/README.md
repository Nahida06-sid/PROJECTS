# Resume Website

## Overview

This project is a personal **Resume Website** built using **HTML5** and **CSS3**. It showcases my profile, skills, learning journey, and projects in a clean, responsive, and user-friendly layout. The project was created to strengthen my understanding of front-end web development fundamentals.

## Features

* About Me section
* Skills section
* Learning Journey section
* Projects section
* Contact Information
* Responsive layout
* Smooth and easy navigation
* Clean and modern user interface

## Technologies Used

* HTML5
* CSS3

## Concepts Implemented

### HTML

* Semantic HTML elements
* Headings and paragraphs
* Navigation links
* Sections
* Lists
* Anchor tags

### CSS

* Selectors
* Box Model
* Flexbox
* Typography
* Colors
* Border Radius
* Box Shadow
* Hover Effects
* Responsive Design

## Purpose

The objective of this project is to build a professional online resume while practicing core HTML and CSS concepts. It also serves as a portfolio piece to showcase my front-end development skills.

## Learning Outcomes

Through this project, I learned how to:

* Structure web pages using semantic HTML.
* Create responsive layouts with CSS.
* Use Flexbox for page alignment.
* Design a clean and user-friendly interface.
* Organize content effectively.
* Improve code readability and maintainability.
