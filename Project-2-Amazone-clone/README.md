# Amazon Homepage Clone

## Overview

This project is a front-end clone of the Amazon homepage built using **HTML5** and **CSS3**. It recreates the layout and design of the Amazon landing page to practice core front-end development concepts such as page structure, Flexbox, positioning, and responsive design.

> **Note:** This project is developed for learning and educational purposes only. It is not affiliated with or endorsed by Amazon.

## Features

* Amazon-inspired navigation bar
* Search bar with category selector
* Hero banner section
* Product category cards
* Shopping panels and promotional sections
* Multi-column footer
* Hover effects for interactive elements
* Responsive layout

## Technologies Used

* HTML5
* CSS3

The purpose of this project is to strengthen my front-end development skills by recreating the homepage of a popular e-commerce website. It helped me understand webpage layout, styling techniques, and responsive design using HTML and CSS.
